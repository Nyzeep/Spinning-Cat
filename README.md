## 香芸与破浪的Kiepei

Kiepei 生活的城市里充满了种种规矩。

这里的一切都有着规定。路灯开关由日照传感器决定，绿植的间距严格遵循城市绿植指南上的数值，连河岸的长椅都标着“建议坐姿”。规矩不仅仅是写在文本、刻在石碑上，同时也长在人们嘴边。“这样做更好”“那样效率更高”——每个人都由衷地这么认为，也由衷地对别人这么说。

Kiepei 从河岸捡回第一袋废弃零件那天，邻居探头看了一眼：“小伙子，你捡的这些东西，一定要琢磨出来好东西，造福一下我们社区呀！”语气像在说“今天天气不错”。后来每次碰面都问创作进度。再后来，城市设计创意辅导员开始上门，带着平板电脑和匹配好的图纸来指导他的创作。

他不讨厌这些。辅导员的图纸确实专业，建议确实合理。只是他偶尔觉得，这座城市的生活就像一部电影，每个人都是剧本里既定的角色，认真地演绎着一部美好但是又隐约差点意思的电影。

Kiepei 创作的过程中总会出现许多次品，这都是一些“不合规的”“不符合创作理念”的废案。  
他时常会看着那些次品，觉得它们反而比一旁的成品多了一份纯粹，但那份纯粹是什么，他也说不清。

他和往常一样在河岸边收拾各种废弃零件。“又是风平浪静的一天。”

他在河岸边四处搜寻散落的废弃零件，将它们一并扔在推车上，脑子里构想着今天的创作方案。忙活了一阵后，他打算把浅滩上一个大破电机捞上来就收工，电机上缠着许多无用的垃圾，需要清理后留下使用价值大的部分。

Kiepei 在处理电机表面附着杂物时，闻到一股清香，那香味是从挂在电机内侧的一个小竹编笼里发出来的。他拆开外壳取下竹笼，发现里面有几株植物，清香就是由它们散发出来的。

这个香气很奇妙，对 Kiepei 来说非常陌生。他从来没在这个城市里闻到过这种气味。回到工作室里，他对着这几株植物端详了很久，也不停地在城市植物库里寻找资料，但都无法确认它们的品种和来源。像这些来历不明的植物就是城市的违规物了——任何未经登记的植物，都应该上交管理局统一审核，如不符合规范，便会销毁。这项严格的植物管理规定确保了居民的健康。

这个味道像魅魔一样勾引着 Kiepei，他心心念念，一直放不下。

Kiepei 查了多天资料，城市植物库翻了无数遍，仍一无所获。后来在社区资料站的一个角落里翻到一本旧得要散架的水文笔记，是一个水文员多年前手写的。里面夹着一张褪色的速写，画的正是这种植物，旁边潦草地注了一行字：“香芸。见于下游远处。水急，船不达。——妖”

“下游远处”是个很模糊的说法。这个城市河流稳浪器覆盖的范围大概五十公里，再往外是什么样，没人说得清。城市手册上只有一句“稳浪区域外水文复杂，不建议非专业船只前往”。偶尔有钓鱼的人划远了，回来都说浪大得吓人，船差点散架。

他萌生了一个想法，但他不想告诉其他人。

隔天创意辅导员上门，带着全新的项目改良方案。Kiepei 看了两秒。“这个方案挺不错，不过实现起来有些难度，我需要花点时间研究一下。”把创意辅导员打发走后，他把门关上，把藏起来的船只改装方案重新放到桌子上钻研。接下来一个多月，Kiepei 白天在工作室里摆弄那些标准零件，偶尔在院子里敲敲打打，邻居路过点头说这回上道了。夜里他把车库门关紧，从后门把河岸捡来的铝板、旧马达、半捆防水布一样一样搬进去。

他要造一艘“违规”的船。

一开始不太顺利。他拿着笔在铝板上画线，画出来的总是标准图纸上的结构——龙骨笔直，船身对称，吃水深度精确到毫米。画完自己看一眼，撕了。重画。又撕。他发现一个问题：他已经不知道怎么画一条“不对”的线了。那些标准结构、制造规则在他脑子里生了根，手一动就往上靠。后面 Kiepei 干脆不画了，直接把铝板架起来，凭感觉切。切歪了也不修，焊歪了也不改。马达是用捡来的旧引擎改装的，原有功率标称早就磨没了，他试了一下——动力这么强！——声音更是大得像航空发动机。

船的样子一言难尽。船身左低右高，焊缝像打过仗的弹痕，马达拼装在船尾看着非常粗犷，防水布绷得皱巴巴的。整艘船蹲在那儿，说好听点叫苏式暴力美学作品。但它有个好处：劲儿大。那台旧马达虽然吵，推力却猛得离谱。Kiepei 试转的时候，整个船身都在抖，像被拽着要往前冲。“力大砖飞可以克服一切阻力。”他自言自语，拍了拍马达壳子。

下水那晚他挑了凌晨一点，按照城市规定，人们在这个点几乎都入睡了。

月亮不大，河面黑漆漆的。稳浪器的嗡鸣声在夜里比白天清楚，像很远的地方有一群蜜蜂。他把船从车库后门推出来，沿着小路往河岸走。轮子没装好，一路嘎吱嘎吱响，他走几步就停下来听一下。到岸边，他把船推进水里。船身歪了一下，吃水比预想的深，但好歹浮住了。

他发动马达。声音确实像航空发动机，在黑夜的河面上炸开，他自己都吓了一跳。赶紧调低油门，船慢慢离开岸边，朝稳浪器边缘的方向滑过去。

水面的纹理开始变了。先是细碎的抖动，然后是小波浪拍打船底，再然后——整艘船像被一只大手从下面托了一下，又重重放下来。很快，Kiepei 就行驶到稳浪区域边缘了。

前面就是“打架的水域”。Kiepei 之前只听人说过，从没见过。月光下能看见水面像开了锅，浪头从不同方向撞在一起，发出低沉的闷响。没有规律，没有方向，就是乱。

他深吸一口气，把油门推上去。船冲进去的第一秒，一个浪从左边砸过来，船身猛地向右歪过去。他整个人差点被甩出去，双手死死抓住船舷。还没等稳住，右边又一个浪顶上来，船又被推向左。

马达在吼。铝板在叫。防水布被风灌得鼓起来，啪啪地响。

Kiepei 刚开始还试着跟浪较劲——船往左歪他就往右压，船往右倒他就往左倾。折腾了十来分钟，胳膊酸得发抖，船还是被浪推来推去。

他忽然想起那几株香芸被浪打翻时，香气反而更浓了。

他松了松手。不跟浪较劲了。船歪过去，他就让它歪。歪到一定角度，船自己会正回来一点。再歪，再正。那台旧马达一直在推，不管船身怎么摆，它只管往前出力。难听点说，这船不是在航行，是在浪里硬拱。

但它在往前走。

不知道过了多久。浪开始变小，从“打架”变成了“推搡”，又从“推搡”变成“摇晃”。马达的声音也不再那么撕心裂肺，慢慢降下来，变成一种稳定的突突声。Kiepei 回头看了一眼。城市的灯光早就不见了。四面都是黑的，只有船头前面一小片水面被月光照得发亮。

又开了大概一小时。空气里忽然飘来一股熟悉的气味。

他把油门收到最小。马达声落下来，周围一下子安静了。水面上飘着那股味道，比他在工作室里闻到的浓得多，也活得多，像是从什么地方大片大片地涌过来的。

船慢慢向前漂。月光照见前方一片岸的影子，不是城市那种石砌的河岸，是土坡，长满了草。再近一点，他看见了岸上铺开一大片植物，月光下叶片的颜色像蒙了一层霜——

香芸。

他把船靠过去。船底蹭到泥，发出一声闷响。马达熄火，周围彻底安静下来。只有风穿过香芸田的声音，沙沙的，像很多片叶子在小声说话。

田垄上有个人。看身形是个少女，蹲在两排香芸中间，手插在土里。她听见动静，站起来转过身，手里还拿着一株刚培好土的香芸苗。

两个人隔着十几步对视了一会儿。

她先开口了。

“你的船好吵哦。”

Kiepei 愣了一下，低头看看自己的船，又抬头看看她。“嗯，对的，那马达……劲儿大。”

她走过来，看了看船身上那些歪歪扭扭的焊缝，又看了看那台旧马达。“这个船看着好奇怪，这是你自己造的吗？”

“又寸。”

“造型挺有趣的。”少女笑了笑。

“能过来就行。”

听到少女的话，Kiepei 心中对造出眼前这艘不合规的另类之船感到一丝得意。

“你是从远方那座城市来的？”

“嗯。”

“头一回有人过来。”她把手里那株香芸苗随手插进田垄边的土里，拍了拍手上的泥。“来干什么呀？”

Kiepei 想了想，从口袋里掏出那个小竹编笼。里面还剩一株香芸，叶子已经有点蔫了。

“我在那边捡到了这玩意，想知道它从哪儿来。”

她看了一眼竹笼，又看了一眼他。

她转身沿着田垄往里面走。Kiepei 跟上去。田垄很窄，两边香芸的叶子擦过他的裤腿，每擦一下就带起一小团香气。她走路很轻快，偶尔会绕一下——不是因为有障碍物，像是特意避开某几株香芸。

“为什么绕开那几株。”

“那几株很害羞，不喜欢人挨太近。不然的话第二天会卷叶子。”

Kiepei 低头看了看。那几株看上去跟别的没什么区别。但他没再问。

田垄尽头是一间矮房子，泥墙草顶，窗户亮着暖黄色的光。门口摆了一排陶罐，种着各种他不认识的植物。她推门进去，不一会儿端了两杯水出来，递给 Kiepei 一杯。水是凉的，带着一点土腥味。Kiepei 喝了一口水。水里的土腥味跟香芸的气味混在一起，居然不违和。

“你一个人住这儿？”

“嗯。还有它们。”她伸手往田里指了指。

“你生活的那个城市……”她朝河对岸的方向晃了一下头，“是不是有很多规矩呀？”

“你为什么会这么觉得。”

“我老师以前讲过。她说河那边有个地方，什么事都有条条框框，种花要按手册来，造船要按图纸来。那儿的人觉得这样最好。”她又补了一句：“我妈妈也提过。她说那边的人都是好人，就是活得有点……累。”

Kiepei 笑了一下。

“我们那儿确实什么都有规矩。”他说。“船怎么造，花怎么种，东西怎么做，都有一套标准。他们说这样最好，最安全。”

“那你觉得呢。”

Kiepei 想了想：“我们的城市按照规矩，一切都井井有条，确实很和谐，那些规矩也确实让人们生活得很安稳。”

“那，你为什么还要造一艘船来这里呢，这艘船看起来就不是很安全，我们那边浪都挺大的。”

Kiepei 看了看手里那株蔫了的香芸，思索了一会。有时我也想看看，规矩之外的东西，或者说是标准答案以外的解法……脑中回荡着这个答案，但他最后没有说出口。

“可能是，纯粹的好奇吧。”

少女没有接话，过了会儿，她弯腰从门口的陶罐里拔出一株很小的香芸苗，根上还带着湿土。  
“给你这个，你那竹笼太小了，住着不舒服。你拿着这个吧，连罐一起。”他接过来。陶罐粗糙，带着土的凉意。

Kiepei 想起一路跟着她来时看到的景象，问到：“你们那边的植物种植规范似乎和我们那边的不一样，你们是可以随意种植作物的吗？”

“嗯……我们这里也有种植规定，但是并不是强制要求要符合一个标准哦。”

“你看这田里。每株隔三步远。”

“为什么。”

“香芸的根会绞在一起。种太近，两株都长不好。”她站起来，拍了拍膝盖上的土。“这不是我定的，是它告诉我的。”

“这算规矩吗。”

她想了一下。

“算吧。但跟你们那边不一样。你们的规矩是人们定好了让你们去合。我们这边是身边所有的事物告诉你的。就像是……”

她停了一下，像是在找一个合适的词。

“叫话。不叫规矩，像那些香芸，就是它们长得不舒服，才让我们去改变种植的方案呀。”

后来 Kiepei 又待了大半天。她带他看了香芸田的排水沟——不是挖出来的，是顺着地势自然形成的。看了她怎么收集雨水，怎么看叶子的颜色判断是不是该施肥。

“这些东西谁教你的。”

“一开始是老师。后来老师走了，就自己看。香芸也会告诉我的。”

“它怎么告诉。”

“渴了叶子就软。缺肥了叶边就黄。虫子多了叶背面有小点。看久了就知道了。”

Kiepei 若有所思。

第二天傍晚，Kiepei 准备和少女告别，他要回程了。

马达发动。船离开泥岸，慢慢转了个弯。她站在田垄上没有挥手，只是站着。身后大片的香芸在傍晚的光里泛着银灰色，像散落在田间的碎月光。

回去的浪还是乱。但 Kiepei 没再跟它较劲。船歪过去正回来，马达只管往前推。来的时候他还会慌，回去的时候他知道了，反倒有些轻松。

稳浪器的嗡鸣声从远处一点点靠近。天亮前 Kiepei 就抵达了城市的河岸。

一段时间后，创意辅导员上门验收 Kiepei 的作品。

“Kiepei 啊，你这次的作品，怎么没有按照上次指导书的方案来呢，这次的大改动和以往的方案都有出入，效果可能会大打折扣吧。”

“不会的，您可以尝试一下这个方案，效果绝对出乎意料。”

“这种设计方式我们闻所未闻，你是怎么知道的呢。”

Kiepei 笑了笑，并没有回答，只是默默望向工作室桌子上摆着的陶罐，上面种着一棵陌生的植物，散发着清香。

“辅导员先生，我想问一下，这几天城市河流稳浪器下游的水文状况怎么样？”

创意辅导员很疑惑，“那段水域常年风浪大作，我们称其为‘打架的水域’，Kiepei 如果近期有出行需求，一定要去船舶管理局申请……”

“我明白了。”送走创意辅导员后，Kiepei 望向了河岸的方向，那里有稳浪器，水面依旧风平浪静。河对岸很远的地方，浪大概还在打架。但他知道了，每个月总有几天，两边的水会累。那时会告诉他，你可以启程了。
